# Dataset Neural Network - Tom Cruise vs Brad Pitt

## Como usar este código

### 1. Instale as dependências:
\`\`\`bash
pip install icrawler
\`\`\`

### 2. Execute o script para baixar o dataset:
\`\`\`bash
python3 baixar_dataset.py
\`\`\`

Isso criará duas pastas com 200 imagens de cada ator:
- dataset_tom_cruise/
- dataset_brad_pitt/" > README.md
